```bash 
brew install nginx
# /opt/homebrew/bin 에 nginx 바이너리 설치됨
```
conf는 /opt/homebrew/etc/nginx 하위에 있다. 

```bash 
brew services start nginx
```
nginx 시작 후 

```bash 
ps -ef | grep nginx
```
nginx 가 코빼기도 안보임 

```bash 
brew services list
```
위 명령어로 nginx 오류 확인함 

결국 bin 설치된 곳에 가서
```bash 
nginx -c /opt/homebrew/etc/nginx/nginx.conf 
// nginx 설정을 etc 하위 nginx.conf로 설정
nginx -s reload
```
로 해결했음.
